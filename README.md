# three-quarters.org

四分之三共生的公開官網。本站用來呈現發起宣言、既有公益現場、治理邊界與參與方式。

## 定位

- `three-quarters.org`：公開網站與可公開的公益說明。
- 私人協會治理工作區：保存內部治理文件，不直接發布。
- `three-quarters.net`：四分之三國際有限公司、產品與生態入口。

本站只收錄經選擇後可對外發布的內容，不會自動同步私人協議、聯絡資料、研究資料或尚未取得同意的客座資訊。

## 技術

純 HTML / CSS，無建置步驟、無登入、無表單、無分析追蹤。由 GitHub Pages 發布。

## 本機預覽

直接開啟 `index.html`，或用任一靜態檔案伺服器預覽。
